## 推荐接入方案

如果你要在 Swift 项目中接入 Apple Music，推荐以下路径：

1. **官方 MusicKit 框架**（iOS 15+ / macOS 12+）：Apple 官方提供的框架，支持搜索、播放、用户音乐库访问等功能。
2. **MusadoraKit**：在官方 MusicKit 基础上提供更多便捷 API。
3. **Cider / AmuseKit**：如果需要直接调用 Apple Music REST API。
4. **UI 参考**：SPStorkController、AlertKit、Apple-Music-UI-Demo 等项目可以作为 UI 实现参考。

### 关键技术点

- **授权**：需要 Apple Developer 账号，在 Certificates 中启用 MusicKit 服务
- **JWT Token**：调用 REST API 需要生成 Developer Token（参考 SwiftJWTSample）
- **用户授权**：使用 `MusicAuthorization.request()` 请求用户音乐库访问权限
- **播放**：使用 `ApplicationMusicPlayer` 播放 Apple Music 中的歌曲
- **搜索**：使用 `MusicCatalogSearchRequest` 搜索歌曲/专辑/艺人
